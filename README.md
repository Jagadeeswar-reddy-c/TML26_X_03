This repository contains the final code for the Adversarial Robustness task in Trustworthy Machine Learning.

## Required Files

Before running the code, keep these files in the same folder:

| File | Description |
| :--- | :--- |
| **train_robust.py** | Main adversarial training script |
| **submission.py** | Submission script (API key pre-configured) |
| **train.npz** | Training data (provided by course — 50,000 images, 9 classes) |
| **model.pt** | Generated by `train_robust.py` |

## Install Requirements

Install the required Python packages using:

```bash
pip install -r requirements.txt
```

## Train the Model

Running this script trains a ResNet18 with PGD adversarial training and saves the best checkpoint to `model.pt`.

### Windows

```bash
python train_robust.py
```

### macOS / Linux

```bash
python3 train_robust.py
```

Expected training time on Apple MPS (M-series MacBook): ~135 minutes for 100 epochs.

## Submit to Leaderboard

After training completes, submit `model.pt` to the evaluation server:

### Windows

```bash
python "submission.py"
```

### macOS / Linux

```bash
python3 "submission.py"
```

## Training Method

The training uses **PGD Adversarial Training** in two phases:

| Phase | Epochs | Method | Purpose |
| :--- | :--- | :--- | :--- |
| 1 | 5 | Clean training | Adapt pretrained weights to `[0,1]` input range |
| 2 | 95 | PGD-3 (ε=8/255, α=2/255) | Adversarial training for robustness |

**Important:** The model is trained on raw `[0, 1]` images (no ImageNet mean/std normalization). The evaluation server passes images in `[0, 1]` format — using ImageNet normalization in the data pipeline causes ~12% clean accuracy on the server.

## Notes
